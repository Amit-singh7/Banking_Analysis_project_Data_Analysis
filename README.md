📊 Banking Data Analytics Project

🔍 Overview
This project focuses on analyzing a banking dataset to extract meaningful insights and support data-driven decision-making. It covers the complete data analytics lifecycle — from data loading and cleaning to visualization and reporting using industry-standard tools.

📁 Dataset
The dataset contains customer-level banking information such as demographics, account details, credit usage, loans, deposits, and risk classifications.

Key Features:
Customer demographics (Age, Gender, Location)
Financial details (Income, Loans, Deposits)
Account information (Savings, Checking, Credit Cards)
Risk and classification metrics

🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn) – Data loading, cleaning, and EDA
SQL Server – Querying and data analysis
Power BI – Interactive dashboard creation
Excel (optional) – Quick validation and checks

⚙️ Project Steps
1. Data Loading
Imported dataset into Python using Pandas
Performed initial inspection (shape, columns, data types)

2. Data Cleaning
Handled missing values
Removed duplicates
Corrected data types
Standardized column names and formats

3. Exploratory Data Analysis (EDA)
Analyzed customer demographics
Studied distribution of income, loans, and deposits
Identified trends, patterns, and outliers
Created visualizations for better understanding

4. SQL Analysis
Loaded cleaned data into SQL Server
Wrote queries to:
Calculate total loans and deposits
Segment customers by risk and income
Analyze customer behavior patterns
Generate aggregated KPIs

5. Power BI Dashboard
Built an interactive dashboard to visualize key metrics:
Total Loans & Deposits
Customer Segmentation
Risk Analysis
Income Distribution
Enabled filters for dynamic insights

📊 Dashboard Highlights
KPI cards for quick insights (Total Customers, Loans, Deposits)
Customer segmentation by income and risk
Trend analysis for financial behavior
Interactive slicers for deeper exploration

📈 Results & Insights
Identified high-value customers based on income and deposits
Analyzed loan distribution across different customer segments
Highlighted potential risk segments using risk weighting
Provided actionable insights for improving customer targeting and financial planning
